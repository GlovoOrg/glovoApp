1. Заказы
Просмотр заказов: Администратор видит список заказов с такими данными, как ID, статус, дата создания, адрес, статус оплаты. Пагинация (10-15 на страницу).
Фильтрация: Фильтрация по статусу (CREATED, IN_PROGRESS, READY_DELIVERING, DELIVERING, DELIVERED), дате создания.
Изменение статуса: Кнопка для изменения статуса заказа на "Взять в обработку" (CREATED → IN_PROGRESS).
Интерфейс: Возможность фильтровать, сортировать, изменять статус и просматривать детали заказа.

2. Заведения
Просмотр заведений: Администратор видит список заведений с названием, статусом, датой создания. Пагинация (10-15 на страницу).
Фильтрация: Фильтрация по названию, статусу.
Добавление и редактирование: Возможность создавать, редактировать, удалять заведения.
Интерфейс: Таблица с заведениями, кнопки для создания, редактирования, фильтрации.

3. Продукты
Просмотр продуктов: Администратор видит список продуктов с названием, ценой, статусом, датой создания. Пагинация (10-15 на страницу).
Фильтрация: Фильтрация по названию, цене, статусу.
Добавление и редактирование: Возможность добавлять, редактировать, удалять продукты.
Интерфейс: Таблица с продуктами, кнопки для добавления, редактирования, фильтрации.

4. Интерфейс Dashboard
Пагинация: 10-15 элементов на страницу.
Сортировка и фильтрация: По статусу, дате, цене и названию.
Интерактивные элементы: Кнопки для изменения статусов, редактирования, удаления.
Детали: Возможность просматривать подробную информацию о заказах, заведениях и продуктах.
